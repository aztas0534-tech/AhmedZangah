## نشر الهجرات على الإنتاج (Supabase)

### المتطلبات
- تم ربط المشروع: `npm run db:link`
- تم تسجيل الدخول إلى Supabase CLI: `npx supabase login`

### نشر الهجرات
```bash
npm run db:push:remote
```

### التحقق
بعد النشر افتح لوحة الإدارة، وإذا ظهر تنبيه “قاعدة البيانات غير محدثة أو لا تطابق الواجهة الحالية” فهذا يعني وجود نقص في الهجرات أو أن قاعدة البيانات المتصلة ليست قاعدة إنتاج المشروع.
